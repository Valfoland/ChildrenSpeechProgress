В данной папке указаны все данные, которые помогают отыскать в ресурсах соответствущие файлы.

Каждые файлы относительно каждой миссии имеют свой формат данных. Так как каждая миссия имеет свои требования относительно подгрузки данных.
В каждой из подпапок Streaming Assets указано какой формат имеют данные.

Примечание: 

1. Для более корректного поиска звуков (т.е. если кто-то опечатался в названии то программа найдет похожий по названию файл)  
в папке JsonDataSounds находится файл, содержащийв себе все имена звуков из папки Sounds. 
Его заполнять вручную не нужно, достаточно перейти в Window/DataLevel указать путь к папке Sounds
программа просмотрит все файлы и сохранит их имена в новый Json файл. 
Данный файл нужно будет переместить в папку JsonDataSounds.

2. В некоторых папках указан ключ = "AppSentence/StartSentence" данные ключи нужны для подгрузки сопроводительных речей в миссиях
